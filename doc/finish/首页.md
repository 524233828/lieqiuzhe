## 首页接口

### 获取用户信息

#### 接口地址

```
GET /index/user_info
```

#### 请求参数

| 字段 | 字段含义 | 字段类型 | 是否必须 |
|:----:|:----:|:----:|:----:|

#### 响应参数

| 字段 | 字段含义 | 字段类型 | 是否必须 |
|:----:|:----:|:----:|:----:|
| avatar | 头像地址 | string | true |

#### 返回示例
````
{
    "code": 1,
    "msg":"success"
    "data":{
        "avatar":""
    }
}
````

### 获取banner列表

#### 接口地址

```
GET /index/banner
```

#### 请求参数

| 字段 | 字段含义 | 字段类型 | 是否必须 |
|:----:|:----:|:----:|:----:|

#### 响应参数

| 字段 | 字段含义 | 字段类型 | 是否必须 |
|:----:|:----:|:----:|:----:|
| list[].img_url | banner图地址 | string | true |
| list[].page_id | 页面ID | int | true |
| list[].url | H5页面的地址，页面ID为1时有用 | string | false |
| list[].params | 页面参数，页面ID不为1时有用 | string | false |


#### 返回示例
````
{
    "code": 1,
    "msg":"success"
    "data":{
        list:[
            {
                "img_url":"https://www.baidu.com/img/bd_logo1.png",
                "page_id":1,
                "url":"https://www.baidu.com"
            },
            {
                "img_url":"https://www.baidu.com/img/bd_logo1.png",
                "page_id":2,
                "params":"333"
            }
        ]
    }
}
````
