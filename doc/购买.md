### 购买用户等级

#### 接口地址

````
POST /buy/user_level
````

#### 请求参数

| 字段 | 字段含义 | 字段类型 | 是否必须 |
|:----:|:----:|:----:|:----:|
| level | 等级 | int | true |
| month | 购买时长 1，3，12选一 | int | true |
| pay_type | 支付方式 wechat，alipay | string | true |

#### 响应参数

| 字段 | 字段含义 | 字段类型 | 是否必须 |
|:----:|:----:|:----:|:----:|

#### 返回示例
````
{
    "code": 1,
    "msg":"success"
    "data":{

    }
}
````
