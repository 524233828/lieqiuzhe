## 首页接口

### 获取banner列表

#### 接口地址

``

#### 请求参数

| 字段 | 字段含义 | 字段类型 | 是否必须 |
|:----:|:----:|:----:|:----:|
| platform | 平台 | int | true |

#### 响应参数

| 字段 | 字段含义 | 字段类型 | 是否必须 |
|:----:|:----:|:----:|:----:|
| [].img_url | banner图地址 | string | true |
| [].type | 点击banner图的类型 0-打开H5页面 1-跳转到APP内模块 | int | true |
| [].url | H5页面的地址或者APP模块参数 | string | true |


#### 返回示例
````
{
    "code": 1,
    "msg":"success"
    "data":[
        {
            "img_url":"https://www.baidu.com/img/bd_logo1.png",
            "type":0,
            "url":"https://www.baidu.com"
        }
    ]
}
````
